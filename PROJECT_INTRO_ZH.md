# 求职助手

求职助手是一个本地优先的个人求职工作台。它帮助用户维护求职画像、按地区扫描岗位、关注目标公司、根据简历和偏好重排推荐、准备申请材料，并完整追踪投递状态。

## 核心问题

求职过程常常分散在招聘平台、公司官网、简历、求职信、笔记和表格里。求职助手把这些动作收进一个每日工作流，同时保留最终申请由用户确认的边界。

## 当前范围

- 多地区个人画像：Singapore、China Mainland、Hong Kong。
- 公司雷达：内置重点雇主目录，也支持自定义官网招聘链接。
- 每日扫描：展示来源进度、失败和受限原因。
- 简历上传与本地优先职业定位分析。
- 5.0 基础评分，叠加方向偏好、地区、城市和关注公司排序。
- Apply / Watch / Drop 决策流。
- 浏览器辅助填表，但停在最终提交前。
- 本地 SQLite 作为数据源。

## 边界

这是一个本地单人产品，不包含账号、云同步或全自动真实申请提交。
